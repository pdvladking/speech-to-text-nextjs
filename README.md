# Speech-to-Text App

A real-time speech-to-text prototype built with **Next.js** and **Tailwind CSS**.
It uses the **Web Speech Api** to capture microphone input and transcribe speech into text instantly - similar to live captions or chat-style interfaces.

- Link [https://speech-to-text-nextjs-pi.vercel.app/]

## Features

- Start/Stop microphone recording
- Real-time transcription displayed in a chat-style UI
- Language support (default: English, Nepali etc.)
- Tailwind CSS styling

## Tech Stack

- [Next.js]
- [Tailwind CSS]
- Web Speech API - Browser-native speech recognition

## Setup & Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/pdvladking/speech-to-text-nextjs.git
   cd speech-to-text-nextjs

2. Install dependencies:
   Bash
   npm install

3. Run the development server:
   Bash
   npm run dev
